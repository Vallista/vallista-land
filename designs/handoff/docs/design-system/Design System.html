Design System

Paper Light 방향의 3-tier 토큰 시스템. 블로그 외 다른 프로덕트(개인 도구, 내부 대시보드)에도 그대로 적용 가능한 foundation + component 라이브러리.

v2.0 approved Pretendard + JBM vanilla-extract ready WCAG 2.1 AA

Rules

  • Body copy는 --ink-2. --ink은 headings·buttons·active items 전용.
  • Divider 기본값은 --line-2. section break처럼 load-bearing 할 때만 --line.
  • Core gray를 컴포넌트에서 직접 참조 금지. role 먼저 정하고 semantic 토큰 추가.
  • Accent는 **링크 · 진행바 · focus ring · CTA · 강조** 5곳에만.

code blocks

Korean-specific rules

✓ Do

한국어 줄바꿈은 단어 경계에서. word-break: keep-all을 전역으로 적용하고 text-wrap: pretty를 같이 쓴다.

✗ Don't

절대 break-all, justify 금지. 음절 단위로 잘리면 본문은 바로 망가진다. text-align: justify도 같은 이유로 하지 말 것.

Motion

3 durations · 4 easings

모션의 목적은 causality 전달. 확인 가능한 반응을 빠르게, 공간 이동을 부드럽게.

120ms
--dur-fast
hover · press · focus
180ms
--dur-base
menu · tab · chip
320ms
--dur-slow
page fade · palette · spring

Easings

TokenCurveUse
--ease-out(0.2, 0.7, 0.3, 1)Appearing elements (default)
--ease-in(0.4, 0, 1, 1)Disappearing elements
--ease-inout(0.4, 0, 0.2, 1)Reordering · sliding
--ease-spring(0.34, 1.56, 0.64, 1)Palette open only. Nowhere else.

Required: prefers-reduced-motion

/* Always include this block */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}

Accessibility

WCAG 2.1 AA
RequirementHow
Focus-visible모든 interactive에 :focus-visible. 2px bg offset + 2px accent @ 55%.
Contrastink-2 on bg → 9.4:1 · ink-3 on bg → 5.3:1 · blue-500 on bg → 5.1:1. 모두 AA.
Headingsh1 → h2 → h3 순서 엄격. 스킵 금지. 시각적 크기 조정은 type scale로만.
KeyboardTab 순서 DOM 순서와 일치. Modal 열리면 focus trap + 첫 입력 포커스. Esc로 닫힘.
Reduced motionprefers-reduced-motion: reduce 시 모든 transition 0.01ms.
Skip link"본문으로 바로 가기" — main 첫 요소로.
Color independence상태를 색만으로 표현 금지. Status chip은 dot + 레이블 + 색 3중.
Dark mode시스템 preference + localStorage 수동 토글 2중. 사용자가 덮어쓸 수 있어야 함.
19

Do & Don't

principles

Color

✓ Do
FeaturedCSSactive

하나의 accent + 4단계 ink + 의미 있을 때만 status 색.

✗ Don't
PurpleOrangeTeal

무지개 태그 팔레트. 카테고리 구분은 타이포 계층으로.

Elevation

✓ Do
hairline shadow only
✗ Don't
border + heavy shadow

Motion

✓ Do
120ms hover / 180ms tab / 320ms palette. Spring은 modal 오픈에만.
✗ Don't
Scroll-triggered 본문 entrance. 무한 애니메이션(shimmer, pulse). Bounce on hover. Parallax.

Iconography

✓ Do
필요한 아이콘만 인라인 SVG로. 1.4 stroke, 16px.
✗ Don't
Lucide/Heroicons 전체 import. 이모지를 UI에 사용. 3색 이상의 글리프.

Data display

✓ Do
의미 있는 숫자 하나. "42 articles", "9 min read". 타이포 계층이 강조를 함.
✗ Don't
data slop — 불필요한 뱃지, 무의미한 통계, 모든 카드에 view count, 채우기용 아이콘.
